# Prompts

A collection of Claude Code skills and workflow prompts.

## Workflows

Factory Factory (FF) pipeline skills for autonomous spec-to-PR workflows:

| Skill | Description |
|-------|-------------|
| [ff](workflows/ff.md) | Full pipeline end-to-end (spec -> implement -> review -> pr) |
| [ff-spec](workflows/ff-spec.md) | Spec-writing stage |
| [ff-implement](workflows/ff-implement.md) | Implementation stage |
| [ff-review](workflows/ff-review.md) | Code review stage |
| [ff-pr](workflows/ff-pr.md) | PR creation stage |
| [ff-resume](workflows/ff-resume.md) | Resume an in-progress pipeline |
| [ff-checkpoint](workflows/ff-checkpoint.md) | Snapshot pipeline progress |
| [ff-status](workflows/ff-status.md) | Show status of active workspaces |
| [use-ff](workflows/use-ff.md) | Usage guide for Factory Factory |
